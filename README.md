# Hierarchy-CLI

Automatic and magic project builder assistant

## Features

 - Easy project generation
 - `flexible` expandable and flexible

## Install
```sh
$ npm install --global hierarchy-cli
```


## Documentation

```sh
hierarchy-cli create --template html-standart newProject
```
